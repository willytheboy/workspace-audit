# Parts Catalog

This folder contains reusable part dossiers extracted from the reviewed applications.

## Structure

- `modals/` for onboarding, settings, and creation flows
- `modules/` for reusable panels, command surfaces, memory/workflow systems, and services
- `scripts/` for analysis and ingestion scripts

Each file records:

- original source path
- why the part is viable
- direct reuse level
- expected adaptation work
- integration target inside `workspace-audit`
